# 🤖 WordCheck Telegram Bot

A powerful Telegram bot for checking spelling and grammar in multiple languages.

## ✨ Features

- ✅ **Spell Checking**: Detect and correct spelling errors
- 📖 **Grammar Suggestions**: Get grammar improvement suggestions  
- 🌐 **Multi-language Support**: 12+ languages including English, Spanish, French, German
- 🎯 **Instant Feedback**: Get results immediately after sending text
- 📊 **Statistics**: Track your writing progress
- 🔒 **Privacy-focused**: Doesn't store your text
